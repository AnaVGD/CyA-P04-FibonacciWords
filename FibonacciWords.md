### Introducción
Las palabras de Fibonacci, F1, F2, F3, ... se definen como:
F<sub>1</sub> = "a"
F2 = "b"
Para cualquier n &geq 3, Fn es el resultado de concatenar Fn-2 con Fn-1.
Las primeras siete palabras de la secuencia de Fibonacci son:
F1 = "a",
F2 = "b",
F3 = "ab",
F4 = "bab",
F5 = "abbab",
F6 = "bababbab",
F7 = "abbabbababbab".
